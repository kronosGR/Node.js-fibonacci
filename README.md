# Fibonacci

## Setup
```
// install express-generator
npm install express-generator

//  express Setup with Handlebars template engine
npx express --view=hbs --git .

// uninstall express-generator
npm uninstall express-generator

// run the Fibonacci service
npm run server

// test the service
curl -f http://localhost:3002/fibonaccci/10

// or with the client 
npm run client
```
